## Birthday Buddy:

1. Read the json data in the data.ts and import it.
   
2. Create a state variable using `useState` hook to store the data.
   
3. Display no of birthdays heading in `App.tsx`
   
4. Create two components `List.tsx` to render users list and `SinglePerson.tsx` to render single user.

5. Iterate over the state variable using `map` method and return single Person in `List.tsx`.
  
6. in `SingePerson.tsx`, destructure the object properties and render in UI.

7. Create a button to clear all in `App.jsx`.

8. On button click, update the state to empty array.
